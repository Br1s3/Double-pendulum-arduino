# Double pendulum on ATmega328p

This is a simple test program to print information on screen with UART redirected on printf.
This program simulates themotion of a double pendulum.

## Ready to start

```console
$ make all
$ make dude
$ screen /dev/ttyACM0 1000000
```

## To quit screen

```concole
$ C-a C-k y
```

## To quit screen

ctrl-a and k and y
